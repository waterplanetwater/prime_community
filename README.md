# Prime Community Website

GitHub Pages 用の静的Webサイトです。

## 構成

```text
prime-community-site/
├─ index.html
├─ styles.css
├─ script.js
└─ assets/
   └─ logo.jpeg
```

## 公開方法

1. GitHubで新規リポジトリを作成
2. このフォルダ内のファイルをすべてアップロード
3. GitHubの `Settings` → `Pages`
4. `Deploy from a branch` を選択
5. `main` / `/root` を選択
6. 表示されたURLへアクセス

## 公開前に変更する場所

`index.html` 内の以下を実際のリンクに変更してください。

```html
https://peatix.com/
```

```html
hello@example.com
```

## ロゴ差し替え

`assets/logo.jpeg` を同名ファイルで置き換えるだけで反映されます。
